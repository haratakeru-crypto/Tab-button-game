# Office UI マスター - URL一覧

## 通常モード

通常のゲームプレイ用URLです。

```
http://localhost:3000/
```

**説明:**
- 通常のゲームモードで動作します
- ターゲットゾーンに正しくクリックすると正解、それ以外は不正解と判定されます
- 正解・不正解のフィードバックが表示されます

---

## デバッグモード

ターゲットゾーンの調整や確認用のデバッグモードURLです。

```
http://localhost:3000/?debug=true
```

**説明:**
- デバッグモードが有効になります
- 画像をクリックすると、クリック座標とターゲットゾーンが表示されます
- ターゲットゾーンの位置（top, left）とサイズ（width, height）を調整できます
- 「修正」ボタンをクリックすると、調整したターゲットゾーンが `questions.json` に保存されます

**デバッグモードの機能:**
- クリック座標の表示（x, yのパーセンテージ）
- ターゲットゾーンの可視化（青色の枠）
- 位置調整（top, leftの数値入力）
- サイズ調整（width, heightのスライダーと数値入力）
- ターゲットゾーンのJSON形式でのコピー
- ターゲットゾーンの保存（修正ボタン）

---

## 使用方法

### 開発サーバーの起動

```bash
npm run dev
```

または

```bash
yarn dev
```

### アクセス方法

1. ブラウザで上記のURLを開く
2. デバッグモードを使用する場合は、`?debug=true` パラメータを追加する
3. 開発サーバーが起動していることを確認（通常は `http://localhost:3000`）

---

## 注意事項

- デバッグモードは開発環境でのみ使用してください
- ターゲットゾーンの修正は `questions.json` に直接反映されます
- 修正後はページが自動的に再読み込みされます
